# Política de Privacidade — PonteVPN 🛡️
*Última atualização: 22 de Abril de 2026*

A PonteVPN está empenhada em proteger a sua privacidade. Esta política descreve como tratamos os seus dados pessoais e, mais importante, o que **NÃO** recolhemos.

## 1. Política de "Zero Registos" (Strict No-Logs)
A nossa infraestrutura foi desenhada para não registar atividade. **NÃO RECOLHEMOS NEM ARMAZENAMOS**:
- Endereços IP (origem ou destino).
- Consultas DNS.
- Conteúdo do tráfego ou sites visitados.
- Horários de conexão ou duração das sessões.

## 2. Dados que Recolhemos (Necessários para o Serviço)
Para operar o serviço, armazenamos apenas o mínimo necessário:
- **Email**: Para gestão da conta, login e comunicações importantes.
- **Dados de Facturação**: Processados pela Stripe. Não guardamos números de cartão no nosso banco de dados.
- **Consumo de Banda (Total)**: Apenas para aplicar limites no plano Free. Este dado é agregado e não está ligado a sessões específicas.
- **Chaves Públicas WireGuard**: Necessárias para estabelecer o túnel. As chaves privadas NUNCA saem do seu dispositivo.

## 3. Conformidade RGPD (Portugal/EU)
Como empresa registada em Portugal, cumprimos integralmente o Regulamento Geral sobre a Proteção de Dados:
- **Direito ao Esquecimento**: Pode apagar a sua conta a qualquer momento, o que remove permanentemente todos os seus dados dos nossos sistemas ativos.
- **Localização dos Dados**: Os dados dos utilizadores da UE são processados em servidores localizados na UE (Hetzner, Alemanha/Finlândia).

## 4. Partilha de Dados
Não vendemos, alugamos ou partilhamos os seus dados com terceiros para fins de marketing. Partilhamos dados apenas com processadores essenciais (ex: Stripe para pagamentos) sob contratos estritos de confidencialidade.
